# Golden Perfume — Laravel (صفحتين + واتساب + تعاليق)

هذا المجلد يحتوي **ملفات جاهزة للدمج** داخل مشروع Laravel جديد.

## الصفحات
- **/** الصفحة الرئيسية: نظرة شاملة على المنتجات + نموذج تعاليق + عرض جميع التعاليق.
- **/products** صفحة المنتجات: عرض العطور و **Le Pack** + سلة طلب قابلة للتعديل + زر إرسال الطلب مباشرة عبر واتساب.

## المتطلبات
- PHP 8.2+
- Composer
- قاعدة بيانات SQLite (موجودة) أو MySQL (اختياري)

## طريقة التثبيت (الأفضل)
1) أنشئ مشروع Laravel جديد:

```bash
composer create-project laravel/laravel golden-perfume-shop
cd golden-perfume-shop
```

2) انسخ كل ملفات هذا المجلد إلى جذر المشروع **واستبدل الملفات عند الطلب**:

- انسخ المجلدات: `app/`, `routes/`, `resources/`, `database/`, `public/`, `config/`
- انسخ ملف: `.env.example`

3) جهّز ملف البيئة:

```bash
cp .env.example .env
php artisan key:generate
```

4) SQLite:
- تأكد أن الملف موجود: `database/database.sqlite` (موجود ضمن الملفات)

5) شغّل المايغريشن والسيدر (لإضافة منتجات تجريبية):

```bash
php artisan migrate
php artisan db:seed
```

6) تشغيل الموقع:

```bash
php artisan serve
```

ثم افتح:
- `http://127.0.0.1:8000/`

## تغيير رقم واتساب
من ملف `.env`:

```env
WHATSAPP_NUMBER=2126xxxxxxxx
```

> ملاحظة: يجب إدخال الرقم **بصيغة دولية** بدون `+` ولا `00`.

## إضافة/تغيير المنتجات
المنتجات موجودة في جدول `products`. يمكنك تعديل Seeder أو إضافة منتجات من قاعدة البيانات.
